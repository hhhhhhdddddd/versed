Versed permet d'éditer plusieurs versions (traduction, interprétation, etc.) d'un texte. Par exemple, on peut partir d'un texte en chinois et éditer sa version pinyin, sa traduction mot à mot, sa traduction ligne par ligne, etc. L'édition est orienté token, un token peut être un mot ou une ligne. L'édition se fait dans une zone d'édition générée à partir des différentes versions. La zone d'édition est divisée en aggrégat de ligne de même numéro. L'aggrégat un est constitué de toutes les lignes numéro 1, l'aggrégat 2 est constitué de toutes les lignes numéro 2, etc. Un aggrégat permet d'éditer les lignes qu'il contient.
